## Changelog for June 2025

We started off the month with a fun launch on [Product Hunt](https://www.producthunt.com/products/open-paper-2) and have been working hard to improve the platform since then.

Broadly, these changes help create a more personalized, efficient, and engaging research experience.

Here are the key changes we've made in June:

### New Features

- [77af49](https://github.com/khoj-ai/openpaper/commit/77af49830b4cc45479b5eea21f48198b40593f7f) Add support for tracking reading status (`todo`, `reading`, `completed`) of papers
- [f6297f](https://github.com/khoj-ai/openpaper/commit/f6297f75de50a0de975cd8e65e574cefa666a1e4) Add search filters to the paper finder, like author, institution, and only Open Access
- [847e32](https://github.com/khoj-ai/openpaper/commit/2e8b8bca7b8f8800c8aedb2684d4a7807d078cd7) Add relevant papers on the home page
- [336a1d](https://github.com/khoj-ai/openpaper/commit/336a1d2c06803150e604076c8dee12aaeba3da5a) Allow customization options when generation an audio overview, incl. focus and style
- [9401fa](https://github.com/khoj-ai/openpaper/commit/a4af04191140cf4fcb271816656c3de3d1bc569d) Add realtime tracking for the paper upload status from client
- [e880d7](https://github.com/khoj-ai/openpaper/commit/e880d727635a85ff57620b741cacbef74880db6c) Add preview image to the uploaded paper for easier lookup
- [9fa6d7](https://github.com/khoj-ai/openpaper/commit/9fa6d7881e76c010c72d8f39e3b77313e8c042ee) Add optional SciHub direct link from the paper finder
- [61cbc3](https://github.com/khoj-ai/openpaper/commit/61cbc32ffa51a86027c1e169d6ffbc11af31b2b8) Add support for AI-generated annotations in the paper viewer to help with skimming more effectively
- [3b5c5e](https://github.com/khoj-ai/openpaper/commit/3b5c5e88cdaaff0d9d947a1e949b9e7e9ccf6827) Add citations to the initial overview that's generated for briefing
- [b0e27b](https://github.com/khoj-ai/openpaper/commit/b0e27b1fe7b63cc67ad47d8305690e92a9f4e2a2) Create a separate microservice that handles paper uploading in separate Celery worker jobs. Track upload status by querying the service and getting status updates.
- [665b6b](https://github.com/khoj-ai/openpaper/commit/665b6b42a135af9bfeb20f9f194f6e3e0b6eb030) Create performance improvements that reduce upload times by ~30% and improve reliability of the upload process
